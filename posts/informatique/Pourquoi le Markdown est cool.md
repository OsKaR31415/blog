Le [markdown](https://docs.framasoft.org/fr/grav/markdown.html) est une syntaxe qui permet de formatter du texte.

Par exemple, ce texte :

```markdown
du **Gras** et de l'*italique*.

une liste de trucs
 - a
 - b
 - c
     - c.1
     - c.2
```

 - intégré dans de nombreuses applications
     - obsidian
     - discord
     - framapad
 - code source lisible
 - exportable
 - compatible
     - latex
     - html 
 - pas de distraction
